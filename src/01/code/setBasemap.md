# Set base map    

By default, the Code Editor map will display with the Google road map. You can change the default basemap type like this:

```js
Map.setOptions('HYBRID');
```
You may choose from four base map types:  

* 'ROADMAP'
* 'SATELLITE'
* 'HYBRID'
* 'TERRAIN'
